## Architecture and integration

A frozen presentation-only projection consumes the existing public editor snapshot, retained source-monitor snapshot, canonical timeline projection, project timeline settings, playback formatting helpers, and live shell-local comparison summary. It converts exact clocks with checked integer arithmetic, applies the core-compatible non-drop or NTSC drop-frame label rules to the continuous display frame, keeps physical playback drops separate from omitted timecode labels, and reports active source identity, source and record ranges, grouping, linking, targeting, synchronization, selection, playback, visual, audio, comparison, degradation, and failure state without accepting commands or editable state. NativeViewport memoizes that role-aware projection through the existing application owner and renders it outside the transformed native frame, so fit, zoom, pan, pixel, fullscreen, cinema, external-display, overlay, comparison, and editorial-feedback behavior remain unchanged.
