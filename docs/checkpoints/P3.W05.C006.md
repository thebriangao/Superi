# P3.W05.C006

## Outcome

Implemented compare, split, wipe, difference, reference, and snapshot presentation views for every native viewer role.

## Architecture and integration

A frozen shell-local comparison contract owns the seven-mode catalog, immutable reference and snapshot captures, exact native surface and frame identity, exact optional source or playback navigation context, bounded split and wipe position, and orientation. `NativeViewport` remains the sole real consumer and keeps the native GPU child mounted while comparison diagnostics and controls follow the same navigation transform as overlays. The editing application owner carries only the formatted program comparison summary across siblings so playback reports live comparison state beside its existing exact temporal, visual, audio, synchronization, and degradation evidence.
