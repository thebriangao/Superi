# P1.W05.C012

## Outcome

Superi now has a managed GPU production path for the existing deterministic CPU wide-gamut reference transform. Canonical premultiplied RGBA16F frames remain GPU resident through transform encoding and ordered submission, while visible results match the CPU reference within the locked normalized absolute-error ceiling.

## Architecture and integration

`GpuWideGamutTransform` derives immutable WGSL constants from `WideGamutTransform`, compiles them through the device-scoped shader cache, and uses explicit sampled-input and storage-output bindings. Encoding validates one single-sample 2D RGBA16F input, allocates a managed RGBA16F output, records one managed compute pass, and returns the output with its retained `GpuPassBatch`. Submission returns the same GPU texture with `GpuPassSubmission`, preserving queue order and exposing the governing fence without adding an ordinary CPU pixel path.
