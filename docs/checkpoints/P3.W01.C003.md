# P3.W01.C003

## Outcome

Integrated the canonical generated TypeScript public API into the production React application
through a transport-neutral `SuperiClient` factory and injected provider.

## Architecture and integration

`createSuperiApiBindings` constructs one generated `SuperiClient` for an injected
`SuperiTransport`, reexports the canonical generated contract, and derives its method, event,
resource, request, and subscription types directly from the generated maps. Its request and
automation entry points share the same typed callable, while event subscriptions and unsubscribe
handles pass through unchanged.

`SuperiApiProvider` memoizes that binding set solely by transport and exposes a nullable React
context so the production bootstrap can remain explicit until C004 supplies the concrete desktop
transport. The integration adds no reconnection, cancellation, error classification, project state,
Tauri command, native engine, or generator behavior, and it preserves the C001 lifecycle client and
C002 EngineControl ownership unchanged.

## Implementation process

The synchronized base was `828722cdc688d106d5a369ce06d75e7231952841`, containing the completed C001
and C002 application seams. A focused contract first failed because the application had no
generated-client adapter or provider. The smallest implementation then added the typed factory,
nullable injected context, production bootstrap boundary, and focused source and runtime contracts
without modifying the canonical generated artifact, its generator, or native application sources.

## Verification

The focused generated-client integration contract passes three cases covering the production
boundary, canonical generated surface, exact promise forwarding, shared request and automation
identity, subscription forwarding, unsubscribe forwarding, and immutable bindings. Strict
TypeScript checking, the production Vite build, and all seven application contracts pass. Binding
determinism remains current, the generated TypeScript API contract passes, and all generator
integration contracts pass without generated or native source changes.
