# P1.W07.C007

## Outcome

Added an automated, offline open-tree dependency-direction gate. The gate fails when a runtime crate
declares an internal normal or build dependency outside the documented DAG, when an unreviewed dev
edge appears, or when a new runtime crate lacks an explicit architecture policy.

## Architecture and integration

`superi-dependency-check` is a repository-only wildcard workspace member, so the live graph check
runs through ordinary workspace tests and is also available as a direct command. It reads Cargo
metadata format version 1 with `--no-deps --locked --offline`, keeps runtime and dev policies
separate, and treats `open/docs/STRUCTURE.md` as the human-readable architecture paired with the
executable policy.

## Implementation process

The verified base was `a11cecdb31fcd706c6163b0dcf0574f8baa5d6c6`. Contract tests were written first and failed because
the checker API was absent. The implementation then added metadata parsing, deterministic
violations, exact reviewed edge policies, the direct command, live-workspace validation, and the
documented contributor workflow.

## Verification

Focused contract tests cover the checked-in workspace, unauthorized normal and build edges,
dev-only edge isolation, and fail-closed handling for unknown runtime crates. The direct command
validated 19 runtime crates and 64 internal edges. Locked default and all-feature workspace builds,
strict all-target workspace Clippy, workspace documentation tests, the workspace suite excluding the
host codec package, and a Rust 1.80 package check all passed from the dedicated checkpoint target.

## Delivery

This record is committed with the substantive checker, its tests, the lockfile update, and the
architecture documentation. The delivery commit containing this record is pushed to `origin/main`;
its SHA is recorded in the matching Google Docs completion description and ignored execution log
after it exists.

## Remaining constraints

New runtime crates or internal edges require an intentional update to both the executable policy and
`open/docs/STRUCTURE.md`. Three `superi-codecs-platform` contract tests could not create macOS
VideoToolbox or AudioToolbox sessions on this host, returning operating-system status values
`-12908` and `1718449215`; the remainder of the workspace suite passed when that package was
excluded. Forbidden network clients and open-tree references to the closed tree are owned by
P1.W07.C008 and remain outside this checkpoint.
