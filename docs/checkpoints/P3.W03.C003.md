# P3.W03.C003 Media import

## Outcome

Superi now imports ordinary media and detected numbered image sequences into the active durable
project from a native picker, native drag and drop, recursive folder scans, the public API, and
local automation. Exact duplicate imports are durable no-ops with no project revision or event.

## Architecture and integration

One bounded `import_media` action extends the existing `superi.project.command.execute` contract
and compound project transaction. The project layer atomically inserts fingerprinted linked-media
references, the API derives exact read permissions and returns typed import evidence, and
`LocalProjectHost` persists the project before returning the existing
`superi.project.state.changed` event. Desktop discovery orders paths, skips symlinks, groups
numbered stills deterministically at the project frame rate, and routes picker, drop, folder, API,
and automation origins through that same authority.
