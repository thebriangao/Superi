# P3.W07.C002

## Outcome

Implemented exact HiDPI observation and dynamic monitor-scale refresh for every primary native
viewer. Scale changes now republish current CSS geometry through the existing Tauri viewport owner,
remain visible in the accessible viewer toolbar, and do not invent monitor, GPU, engine, or
persistence state.

## Architecture and integration

`app/src/display-scale.ts` owns one pure immutable display-scale observation. It validates a bounded
finite device-pixel ratio, suppresses duplicate resize signals, watches window and visual viewport
resize, and re-arms the exact resolution media query after each accepted scale transition. Invalid
dynamic values preserve the last valid observation, while an invalid initial environment fails
before installing listeners.

`NativeViewport` retains its existing CSS geometry, role, visibility, analysis, external-display,
color, and Tauri command owners. It consumes only the observed scale factor, republishes through
the existing animation-frame-coalesced placement callback, and shows the current scale plus source
revision without changing viewer zoom, project state, frame pixels, or monitor selection.

## Implementation process

The focused contract first failed because the display-scale module did not exist. Implementation
added the browser adapter, deterministic observer, accessible scale output, and two exact placement
uses. The existing resize and visibility listeners remain intact, so ordinary layout changes and
dynamic resolution transitions converge on the same bounded publisher.

The complete frontend suite passes 175 tests, strict TypeScript passes, the production build

## Remaining constraints

The browser device-pixel ratio remains the CSS-to-physical scale authority supplied to Tauri; the
native viewport owner still validates and applies physical placement. Physical multi-monitor
certification remains platform-lane work, and this checkpoint does not change monitor identity,
ICC policy, frame rendering, engine diagnostics, workspace persistence, or user-authored state.
