# P3.W05.C004

## Outcome

Implemented safe-area, guide, grid, ruler, center, aspect, and custom overlays for every native viewer role.

## Architecture and integration

A frozen presentation-only catalog owns the seven overlay definitions, immutable visibility state, ordered projection, and deterministic custom inset geometry. NativeViewport renders the selected overlays as pointer-transparent layers above the same transformed frame, preserving C003 fit, zoom, pan, pixel, fullscreen, cinema, image-fidelity, comparison, temporal, and external-display intent. No playback, timecode, or status-display state enters the overlay contract, leaving C005 and later checkpoints untouched.

## Implementation process

Added the focused overlay contract first and observed the missing-module failure. Implemented the smallest pure state owner, accessible per-viewer visibility controls, deterministic overlay geometry, transformed presentation layer, real editing-workspace integration assertions, and current map evidence.
