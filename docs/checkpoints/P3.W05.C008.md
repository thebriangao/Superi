## Outcome

Implemented exact Program viewer transform controls backed by ordinary typed graph parameters.

## Architecture and integration

A frozen projection resolves exactly one current selected clip into its timeline-scoped graph, validates the canonical graph envelope, schema references, node and port topology, node order, parameter drivers, and the exact built-in `superi.effect.transform` 1.0.0 contract, then decodes every matrix scalar and sampling choice without acquiring authored-state ownership. `NativeViewport` renders every reachable transform in canonical graph order outside the navigated frame, keeps driver-owned values inspectable and read-only, and publishes one ordered `mutate_graph` action through the existing revision-fenced application project owner. Viewer fit, zoom, pan, pixel, cinema, fullscreen, overlays, comparison, exact status, native placement, and engine-owned rendering remain separate from authored transform state.
