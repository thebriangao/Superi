# P1.W07.C005

## Outcome

Added a blocking Tauri Rust workflow that formats, tests, lints with warnings denied, and compiles a
native desktop host on current macOS arm64, macOS Intel, Windows, and Ubuntu runner families.

## Architecture and integration

The dedicated `.github/workflows/tauri.yml` workflow owns the desktop-shell gate without changing
the open Rust workspace or frontend workflow. The pinned `ci/tauri-smoke/` Tauri 2 host is a real
native wry build and a mock-runtime command consumer, but remains explicitly separate from the
future Phase 3 application and its editor state.

Linux installs the documented WebKitGTK 4.1 and desktop integration headers. All platforms use an
immutable checkout action, read-only repository permissions, locked Cargo resolution, and the same
four mandatory Rust commands without soft-failure behavior.

## Implementation process

The verified base was `6f69a57e478a1cfe7fa6db8f0173aedfe27c6c5b`. Contract tests first failed
on the absent Tauri manifest and workflow. The first native compile then exposed an invalid
production-icon assumption, and the next test compile exposed Tauri's explicit mock-runtime feature
gate; both findings were corrected by constructing the real native builder without launching it and
enabling only the required `test` and `wry` features.

## Verification

Fresh focused proof passed the Node workflow contracts, Rust formatting, the mock-runtime unit test,
strict all-target Clippy, and native macOS compilation with
`CARGO_TARGET_DIR=/private/tmp/superi-p1-w07-c005-target`. Fresh widening proof passed open-workspace
formatting, the boundary and dependency-direction tools, the locked workspace build and test suite,
strict all-target workspace Clippy, documentation tests, and validation of all 10 fixture versions
and 23 payloads.

## Delivery

The workflow, pinned smoke host, lockfile, contracts, maps, and this record are delivered together
by the commit containing this file. The exact delivered commit is recorded in the matching Master
Build Checklist completion description after `origin/main` containment is confirmed.

## Remaining constraints

The smoke host proves native Tauri source compatibility and command registration, not the Phase 3
React application, native viewport, public API transport, editorial behavior, or physical hardware
lanes. Hosted Windows and Linux execution remains the authoritative proof for their native SDK and
linker paths after this commit reaches GitHub Actions.
